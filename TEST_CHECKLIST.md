# 情绪气泡 - 自测 checklist

每次代码修改后，按以下清单逐项验证：

## 基础功能测试

### 首页
- [ ] 页面正常加载，标题显示"情绪气泡"
- [ ] 主题选择器可点击（治愈系/赛博朋克/极简白）
- [ ] 时间选择器可点击（1分钟/3分钟/5分钟）
- [ ] "开始"按钮可点击
- [ ] "查看历史"按钮可点击

### 游戏过程
- [ ] 点击开始后进入3秒倒计时
- [ ] 倒计时结束后显示气泡
- [ ] 气泡可以点击/戳破
- [ ] 戳破时有音效和粒子效果
- [ ] 进度条正确显示剩余时间
- [ ] 计时器正确倒计时

### 结果页
- [ ] 时间结束后显示结果页
- [ ] 显示戳破的情绪标签
- [ ] 显示AI建议
- [ ] "再来一轮"按钮可点击
- [ ] "关闭"按钮可点击 → 返回首页 ✓

### 首页返回（重点回归测试）
- [ ] 点击"关闭"后回到首页 ← **上次修复**

### 历史记录
- [ ] 可以查看历史记录
- [ ] 可以清空历史
- [ ] 可以返回首页

## 边界情况
- [ ] 快速连续点击气泡
- [ ] 切换不同主题
- [ ] 选择不同时间
