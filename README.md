# 念起 (NianQi)

> 念起即觉，觉已不随

## 🎮 项目简介

**念起**，取自禅宗智慧"念起即觉，觉已不随"——当念头升起时觉察它，觉察后便不再执著。

这是一个交互式情绪觉察小游戏。漂浮的气泡如同心中升起的念头，轻轻触碰，让它们自然消散。在这个过程中，探索内心深处的感受，获得觉知与平静。

## ✨ 功能特性

### 核心玩法
- 🎯 **戳破气泡** - 点击屏幕上漂浮的情绪气泡将其戳破
- ⏱️ **时长选择** - 提供 1 分钟 / 3 分钟两种模式
- 🔚 **随时结束** - 游戏中可点击右上角按钮提前结束
- 🌱 **深层探索** - 戳破气泡会生成更细分的子情绪，支持 3 层深度挖掘

### 情绪交互系统

| 机制 | 说明 |
|------|------|
| 🔄 **情绪消散** | 戳破正向情绪后，对立的负向情绪气泡会渐隐飘散 |
| ✨ **情绪共振** | 相似情绪的气泡会发光放大，产生共鸣效果 |
| ⚖️ **情绪平衡** | 系统检测到情绪偏向时，自动补充对立情绪供探索 |
| 🎲 **智能随机** | 去重、类别均衡、上下文感知的情绪生成算法 |

### 情绪库 (350+ 情绪词)

**正向情绪**
- 开心类：窃喜、心动、神清气爽、春风得意、心花怒放...
- 幸福类：小确幸、满足、人间值得、此生足矣...
- 平静类：释然、云淡风轻、心如止水、岁月静好...
- 勇气类：豁出去了、破釜沉舟、义无反顾、干就完了...
- 感恩类：谢谢你、幸亏有你、遇见你真好、何其有幸...

**中性/复杂情绪**
- 想念类：念念不忘、魂牵梦绕、日思夜想...
- 矛盾类：爱恨交织、五味杂陈、进退两难...
- 执念类：不甘心、不撞南墙不回头...
- 倔强类：不服输、不认命、硬撑...

**负向情绪**
- 难过类：心碎、黯然神伤、痛彻心扉...
- 焦虑类：如履薄冰、提心吊胆、压力山大...
- 落寞类：被遗忘、人走茶凉、曲终人散...
- 脆弱类：快撑不住了、一碰就碎、装作没事...

### 视觉主题

| 主题 | 风格 |
|------|------|
| 🧘 治愈系 | 温暖柔和的紫蓝渐变 |
| 🌲 森林 | 清新自然的绿色系 |
| 🌅 日落 | 夕阳西下的橙红渐变 |

### 音效体验
- 🎵 **自然背景音** - 溪流声、轻风声、深沉环境音、偶尔的鸟鸣
- 💥 **戳破音效** - 三种随机音效，每次都有惊喜
- 🔔 **状态音效** - 开始、结束的提示音

### 气泡物理
- 🌫️ **自然漂浮** - 气泡在视口内随机漂浮
- 💫 **碰撞弹开** - 气泡之间有物理碰撞效果
- 🌑 **生命周期** - 气泡 8-12 秒后自然消散
- ✨ **粒子特效** - 戳破时绽放粒子和扩散圈

### 其他功能
- 📊 **结果分析** - 统计戳破的情绪，生成 AI 建议
- 📖 **历史记录** - 查看过往的情绪释放记录
- 💾 **CSV 导出 / 导入** - 在 **设置** 页选好「本周 / 本月 / 全部」后 **导出 CSV** 或 **导入 CSV**（合并或替换）。导入后会按记录 **重算** 依赖累计数据的成就；时段/单次练习类成就 CSV 无法还原，**已解锁的会保留**。**清空历史** 也在设置页
- 📱 **移动适配** - 支持触摸操作、响应式布局；小屏首页**不展示主题圆点**（更简洁），靠**整页左右滑动**切换治愈 / 森林 / 日落；宽屏仍显示主题圆点可点选

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/Lotus1717/emotion-bubbles.git
cd emotion-bubbles

# 方法1: 使用 npx serve
npx serve .

# 方法2: 使用 Python
python3 -m http.server 8080
```

访问 http://localhost:8080

### 在线体验

🌐 **GitHub Pages**: https://Lotus1717.github.io/emotion-bubbles/

### 添加到主屏幕（图标）

- **iOS Safari**：依赖 `<link rel="apple-touch-icon" href="…">`（建议 **180×180** 正方形 **PNG**）。已写在 `index.html`。
- **Android Chrome**：依赖根目录 `site.webmanifest` 里的 `icons`（常用 **192×192**、**512×512**）。需通过 **HTTPS** 或 **localhost** 访问，manifest 才能被正常读取。
- 替换图标时：同步更新 `nianqi-icon.png`（或改路径），并保证 `manifest` 与 `apple-touch-icon` 指向同一套资源；白底可跑 `python3 scripts/make-icon-transparent.py nianqi-icon.png`；大图可缩放到 512 并压缩：`python3 scripts/optimize-icon.py`。

## 📁 项目结构

```
emotion-bubbles/
├── index.html              # 主页面入口
├── site.webmanifest        # PWA /「添加到主屏幕」名称与图标（Android）
├── css/
│   └── styles.css          # 样式表（动画、主题、响应式）
├── js/
│   ├── app.js              # 应用入口，DOM 初始化与事件绑定
│   ├── game.js             # 游戏主控制器，状态管理
│   ├── bubble.js           # 气泡管理器，创建/销毁/动画
│   ├── emotions.js         # 情绪数据库，颜色/层级/关系映射
│   ├── physics.js          # 物理引擎，碰撞检测/边界弹回
│   ├── audio.js            # 音频管理，Web Audio API
│   ├── suggestions.js      # AI 建议模板
│   ├── historyBackup.js    # 情绪记录 CSV 解析与导出（设置页；内含 JSON 工具函数供测试）
│   └── constants.js        # 配置常量
├── README.md               # 项目说明
├── TEST_CHECKLIST.md       # 测试清单
└── test.sh                 # 测试脚本
```
