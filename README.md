# 情绪气泡 (Emotion Bubbles)

> 释放你的情绪，在戳破气泡中获得内心的平静

## 🎮 项目简介

在繁忙的生活中，我们常常压抑自己的情绪。**情绪气泡**是一个交互式 Web 小游戏，让你通过戳破气泡的方式释放和疏导负面情绪，获得内心的平静与治愈。

## ✨ 功能特性

### 核心玩法
- 🎯 **戳破气泡** - 点击屏幕上漂浮的情绪气泡，将其戳破
- ⏱️ **计时挑战** - 提供 1分钟 / 3分钟 / 5分钟 三种时长选择
- 🏷️ **情绪标签** - 戳破气泡后显示对应的情绪标签
- 🌱 **情绪挖掘** - 点击情绪气泡会生成更细分的子情绪

### 情绪交互逻辑 🔥 核心特色
- 🔄 **对立消失** - 点击某个情绪气泡后，相反情绪的气泡会渐隐消失
- ➕ **相近保留补充** - 相近情绪的气泡会保留，并在附近补充新气泡
- 🎨 **流畅体验** - 戳破后立即在附近补充新气泡，体验丝滑

### 情绪库 📚
项目内置了丰富的情绪标签，涵盖：
- 基础情绪：开心、难过、愤怒、恐惧、惊讶、恶心、羞耻、焦虑...
- 复杂情绪：迷茫、疲惫、孤独、失落、压抑、烦躁、委屈、怀念...
- 治愈情绪：温暖、感动、治愈、平静、释然、小确幸、满足...
- 社恐情绪：尴尬、脚趾抠地、社死、想消失、想钻地缝...
- 职场情绪：不想上班、卷不动了、倦了、烦死了、周一综合症、想躺平...
- 还有更多细分情绪...

### 视觉主题 🎨
| 主题 | 风格 |
|------|------|
| 🧘 治愈系 | 温暖柔和的紫蓝渐变 |
| 🌲 森林 | 清新自然的绿色系 |
| 🌅 日落 | 夕阳西下的橙红渐变 |
| 🌙 静谧蓝 | 深色系放松氛围 |

### 音效体验 🔊
- 🎵 **背景音乐** - 舒缓的 ambient 轻音乐 / 篝火白噪音
- 💥 **戳破音效** - 三种随机音效风格，每次戳破都有惊喜

### 气泡动画 ✨
- 🌫️ **视口内漂浮** - 气泡在视口内自然漂浮移动
- 💫 **物理碰撞** - 气泡之间有碰撞弹开效果
- 🌑 **生命周期** - 气泡会缓慢变透明后消失
- ➰ **平滑过渡** - 优化了淡入淡出效果

### 其他功能
- 📖 **历史记录** - 查看过往的情绪释放记录
- 🧹 **清空历史** - 一键清除所有历史记录
- 📱 **移动端优化** - 支持触摸操作，响应式布局

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/Lotus1717/emotion-bubbles.git
cd emotion-bubbles

# 方法1: 使用 npx serve
npx serve .

# 方法2: 使用 Python
python3 -m http.server 8080
```

然后访问 http://localhost:8080
