# 情绪气泡 (Emotion Bubbles)

一个帮助用户释放情绪的交互式 Web 小游戏。

## 🎮 项目简介

在繁忙的生活中，我们常常压抑自己的情绪。情绪气泡让你通过戳破气泡的方式来释放和疏导负面情绪，获得内心的平静。

## ✨ 功能特性

### 核心玩法
- 🎯 **戳破气泡** - 点击屏幕上漂浮的情绪气泡，将其戳破
- ⏱️ **计时挑战** - 提供 1分钟 / 3分钟 / 5分钟 三种时长选择
- 🏷️ **情绪标签** - 戳破气泡后显示对应的情绪标签

### 情绪交互逻辑
- 🔄 **对立消失** - 点击某个情绪气泡后，相反情绪的气泡会渐隐消失
- ➕ **相近保留** - 相近情绪的气泡会保留，并在附近补充新气泡
- 🎨 **流畅体验** - 戳破后立即在附近补充新气泡，体验更流畅

### 视觉主题
- 🧘 **治愈系** - 温暖柔和的配色
- 🌲 **森林** - 清新自然的绿色系
- 🌅 **日落** - 夕阳西下的渐变色彩
- 🌙 **静谧蓝** - 深色系放松氛围

### 音效体验
- 🔊 **戳破音效** - 三种随机音效风格
- 🎵 **背景音乐** - 舒缓的 ambient 轻音乐 / 篝火白噪音

### 其他功能
- 📖 **历史记录** - 查看过往的情绪记录
- 🧹 **清空历史** - 清除所有历史记录
- 📱 **移动端优化** - 支持触摸操作
